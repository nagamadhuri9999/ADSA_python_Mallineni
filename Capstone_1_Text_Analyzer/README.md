# Capstone Project 1: The Smart Text & Anagram Analyzer

## Overview
This capstone project ties together everything you've learned from **Day 1 to Day 4**:
- Variables and Data Types
- If/Else Control Flow
- While and For Loops
- Lists and String Methods
- Functions and Scope
- Recursion

## The Application
You will build a Command Line Interface (CLI) application that allows a user to:
1. Input a block of text.
2. Clean the text (remove punctuation, lowercase it).
3. Count the frequency of specific words.
4. Check if two phrases are anagrams.
5. Play a recursive "Guess the Anagram" mini-game.

## Files
- `app.py`: The complete, working application. Students can run this to see how a full Python script is structured.
- `app_template.py`: A boilerplate version with empty functions and `TODO` comments. Students must fill in the blanks to make the app work!

## How to use
Have students run `app.py` first to see the final product. Then, have them open `app_template.py` and implement the logic themselves!
