# Day 12: Circular Queues and Deques Quiz

**1. In a circular queue of capacity `C`, how do we increment the `rear` pointer to insert a new element?**
a) `rear = rear + 1`
b) `rear = (rear + 1) % C`
c) `rear = (rear % C) + 1`
d) `rear = rear % C`

**2. What is the primary advantage of a Circular Queue over a Linear Queue?**
a) It requires less memory.
b) It is faster in basic operations.
c) It reuses empty spaces created by dequeued elements.
d) It allows inserting from both ends.

**3. Which data structure supports insertion and deletion at both ends?**
a) Priority Queue
b) Circular Queue
c) Stack
d) Deque

**4. What is the time complexity of pushing an element to the front of a Deque?**
a) O(1)
b) O(n)
c) O(log n)
d) O(n^2)

**5. Which of the following is NOT an application of Deque?**
a) Palindrome Checker
b) Undo operations in text editors
c) Multi-processor scheduling (work-stealing)
d) Sliding Window Maximum

---
*Answers:*
*1: b, 2: c, 3: d, 4: a, 5: b (Undo operations are typically managed with a Stack)*
