# Day 12: Task

## Objective
To master the implementation and application of Circular Queues and Deques.

## Tasks
1. **Implement `MyCircularQueue`**: Complete the implementation of the Circular Queue using arrays in Python. Ensure all edge cases (full queue, empty queue) are covered.
2. **Implement `MyCircularDeque`**: Complete the LeetCode problem "Design Circular Deque". Test your methods manually to understand how the front and rear pointers change dynamically.
3. **Sliding Window Problem**: Review the code for `sliding_window_maximum` in `Day12_Practice_Problems.py`. Write down dry-run iterations on paper using the input array `[1, 3, -1, -3, 5, 3, 6, 7]` and `k = 3`. 
4. **Take the Quiz**: Answer the questions in `Day12_Quiz.md` to reinforce your theoretical knowledge.

## Submission
Ensure your `MyCircularQueue.py` and `Day12_Leetcode.py` run without errors and pass basic test cases. Push your completed files to the repository.
