Day 1 Notes: Python Fundamentals, Variables, Data Types, and Operators

1. What is Python?
------------------
- Python is a high-level, interpreted programming language known for its readability and simplicity.
- It was created by Guido van Rossum and released in 1991.
- Real-world example: Instagram uses Python (specifically the Django framework) to handle millions of user requests and data transactions seamlessly.
- Key features:
  - Interpreted: Code is executed line-by-line, making debugging easier.
  - High-level: Abstracts away complex details like memory management.
  - Readable: Uses English-like syntax and indentation instead of curly braces.

2. Why choose Python?
---------------------
- Beginner-friendly: Simple syntax reduces the learning curve.
- Versatile: Used in web development, data analysis, artificial intelligence, automation, and more.
- Huge ecosystem: Rich set of libraries (e.g., Pandas for data analysis, TensorFlow for AI).
- Real-world example: A data analyst at a retail company can write a 10-line Python script to combine 50 Excel spreadsheets automatically, saving hours of manual work.

3. Basic Python Syntax
----------------------
- Statements: A single line of code is generally a single statement.
- Indentation: Python uses spaces (usually 4) to define code blocks (e.g., inside an `if` statement or a function), unlike other languages that use `{}`.
- Comments: Start with `#`. Used to explain code.
- Real-world example: Adding a comment `# Calculate the discounted price for VIP customers` before a complex math calculation helps other developers understand the intent.
  ```python
  # This is a comment
  print("Welcome to Python Analytics!")
  ```

4. Variables
------------
- A variable is a named container that stores data in the computer's memory.
- In Python, you create a variable by assigning a value using the `=` operator.
- Real-world example: Think of a variable like a labeled box in a warehouse. If the box is labeled `customer_age`, you know the value inside is the age of the customer.

Naming rules:
- Use letters, numbers, and underscores (`_`).
- Cannot start with a number (e.g., `1st_name` is invalid, but `name_1` is valid).
- Cannot use Python reserved keywords (e.g., `if`, `for`, `class`).
- Best practice: Use `snake_case` (e.g., `total_sales_amount`).

Real-time Example:
```python
# Storing real-time weather sensor data
sensor_location = "Downtown Seattle"
current_temperature_f = 68.5
is_raining = False
```

5. Data Types
-------------
Python infers the type of data automatically (dynamic typing). Here are the primary built-in data types:

Numeric Types:
- `int` (Integer): Whole numbers.
  - Real-world use: Counting the number of items in a shopping cart (`items_count = 5`).
- `float` (Floating-point): Decimal numbers.
  - Real-world use: Storing a product's price or a user's account balance (`account_balance = 1250.75`).

String Type (`str`):
- Text values enclosed in single (`'`) or double (`"`) quotes.
- Real-world use: Storing a user's email address (`user_email = 'alice@example.com'`).

Boolean Type (`bool`):
- Represents `True` or `False`.
- Real-world use: Flagging if a user has admin privileges (`is_admin = True`).

Collection Types:
- `list`: Ordered, mutable (changeable) collection of items.
  - Real-world use: A list of recent transaction amounts.
  - `transactions = [12.50, 45.00, 100.99]`
- `tuple`: Ordered, immutable (unchangeable) collection.
  - Real-world use: Storing fixed geographic coordinates (latitude, longitude).
  - `headquarters_location = (40.7128, -74.0060)`
- `dict` (Dictionary): Key-value pairs for structured data.
  - Real-world use: Storing a customer's profile.
  - `customer = {"name": "John", "loyalty_points": 450}`
- `set`: Unordered collection of unique items.
  - Real-world use: Finding unique visitors to a website from a list of IP addresses.

Type checking and conversion:
```python
# Converting a string input from a web form into a float for calculation
user_input = "25.50"
discount = float(user_input)
```

6. Operators
------------
Operators allow you to manipulate data and variables.

Arithmetic Operators:
- `+` (Add), `-` (Subtract), `*` (Multiply), `/` (Divide)
- `//` (Integer Division): Drops the decimal part.
- `%` (Modulo): Returns the remainder.
  - Real-world use: Determining if a number is even or odd (`number % 2 == 0`) or batching tasks by 10s.

Comparison Operators (returns a Boolean):
- `==` (Equal), `!=` (Not equal), `>`, `<`, `>=`, `<=`
- Real-world use: Checking if a user's password matches the saved password (`input_pwd == saved_pwd`).

Logical Operators:
- `and`: True if both are true.
- `or`: True if at least one is true.
- `not`: Inverts the boolean.
- Real-world use: Determining loan eligibility.
  - `is_eligible = (credit_score > 700) and (income >= 50000)`

Membership Operators:
- `in`, `not in`
- Real-world use: Checking if a searched item is in stock.
  - `"Laptop" in available_inventory`

7. Practical Mini Project: Sales Analysis Script
------------------------------------------------
Let's tie it all together with a real-time data analysis example. Imagine you are analyzing sales data for a small coffee shop.

```python
# 1. Variables and Data Types
shop_name = "Brew & Byte"              # str
coffee_price = 4.50                    # float
is_open = True                         # bool

# 2. Collections (Lists and Dictionaries)
# Daily sales quantities for Monday to Friday
daily_sales_cups = [120, 145, 130, 160, 200]
menu_prices = {"espresso": 3.00, "latte": 4.50, "cappuccino": 4.00}

# 3. Arithmetic Operators
total_cups_sold = sum(daily_sales_cups)
average_cups_per_day = total_cups_sold / len(daily_sales_cups)

# Calculate estimated total revenue assuming average price
estimated_revenue = total_cups_sold * coffee_price

# 4. Logical & Comparison Operators
# Did we meet our weekly goal of 700 cups?
goal_met = total_cups_sold >= 700

# 5. Outputting the results
print("--- Sales Report for", shop_name, "---")
print("Total Cups Sold:", total_cups_sold)
print("Average Cups/Day:", average_cups_per_day)
print("Estimated Revenue: $", estimated_revenue)
print("Weekly Goal Met?", goal_met)
```

8. Practice Questions & Interview Prep
--------------------------------------
1. How would you explain Python to someone who has never programmed before?
   - Answer: It's a tool that lets you give instructions to a computer using clear, almost English-like sentences, making it great for automating tasks or analyzing data.
2. What are the basic syntax rules you must follow when writing Python code?
   - Answer: You must use consistent indentation (spacing) to group code blocks, avoid missing colons after control statements, and use `#` for comments.
3. How do you declare variables in Python, and how are they different from variables in other languages?
   - Answer: You simply use the `=` sign (e.g., `age = 30`). Unlike languages like Java or C++, you don't need to declare the data type beforehand (dynamic typing).
4. What built-in data types does Python provide, and when would you use each?
   - Answer: `int`/`float` for math, `str` for text, `bool` for conditions, `list` for ordered items (like a to-do list), and `dict` for key-value pairings (like a user profile).
5. What categories of operators exist in Python? Give examples of each.
   - Answer: Arithmetic (`+`, `-`), Comparison (`==`, `>`), Logical (`and`, `or`), and Membership (`in`).
