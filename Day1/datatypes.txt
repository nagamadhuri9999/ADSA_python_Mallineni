# Detailed Notes: Python Data Types

Python infers the type of data automatically. The data type determines what operations can be performed on the data (e.g., you can multiply numbers, but you cannot multiply two strings).

## 1. Numeric Types
Used to store numerical values for mathematical operations.
- **`int` (Integer):** Whole numbers, positive or negative, without decimals.
  - *Real-world use:* Counting the number of items in a shopping cart (`items_count = 5`).
- **`float` (Floating-point):** Numbers containing a decimal point.
  - *Real-world use:* Storing a product's price, interest rates, or GPS coordinates (`price = 19.99`).

## 2. String Type (`str`)
Used to store text. Strings must be enclosed in either single quotes (`'`) or double quotes (`"`).
- *Real-world use:* Storing user inputs, names, emails, or file paths (`user_email = 'alice@example.com'`).
- **Concatenation:** You can combine strings using the `+` operator.
  ```python
  greeting = "Hello, " + "World!"
  ```

## 3. Boolean Type (`bool`)
Represents one of two values: `True` or `False`. Used heavily in logic and decision-making (control flow).
- *Real-world use:* Flagging states like user permissions or toggle switches (`is_admin = True`, `is_payment_successful = False`).

## 4. Collection Types
Used to store multiple items in a single variable.

- **`list`:** An ordered, mutable (changeable) collection of items. Defined using square brackets `[]`.
  - *Real-world use:* Storing a sequence of daily sales figures or a queue of tasks.
  - `transactions = [12.50, 45.00, 100.99]`
- **`tuple`:** An ordered, immutable (unchangeable) collection. Defined using parentheses `()`.
  - *Real-world use:* Storing data that should never change, like geographic coordinates (latitude, longitude) or RGB color values.
  - `location = (40.7128, -74.0060)`
- **`dict` (Dictionary):** An unordered collection of Key-Value pairs. Defined using curly braces `{}`. Fast lookup.
  - *Real-world use:* Storing structured profiles, configurations, or JSON-like data.
  - `customer = {"name": "John", "loyalty_points": 450}`
- **`set`:** An unordered collection of unique items. Duplicates are automatically removed. Defined using curly braces `{}`.
  - *Real-world use:* Keeping track of unique visitors by IP address or removing duplicates from a list.
  - `unique_ips = {"192.168.1.1", "10.0.0.5"}`

## 5. Type Checking and Conversion
You will often need to know what type a variable is, or convert it to another type (type casting).
- **Checking Type:** Use the `type()` function.
  ```python
  x = 5.5
  print(type(x))  # <class 'float'>
  ```
- **Conversion:** Use built-in functions like `int()`, `float()`, `str()`, `bool()`.
  ```python
  # Converting user input (which is always a string) into a number
  user_input = "25"
  age = int(user_input)
  ```
