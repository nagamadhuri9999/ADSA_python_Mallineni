Day 1: Python Fundamentals for ADSA (6-hour online class)

Session goal:
Introduce Python fundamentals, build confidence with syntax, variables, data types, and operators, and connect these concepts to a simple analytics-style exercise.

Class structure:

1. Welcome and course introduction (30 minutes)
- Brief overview of ADSA and why Python is the best choice
- Course objectives and learning outcomes
- Tools setup check: Python installation, IDE (VS Code/IDLE), and running a simple script

2. What is Python? (30 minutes)
- Definition and history of Python
- Key features: interpreted, high-level, readable
- Why Python is used for data analysis, automation, and web apps
- Interview-style question: "How would you explain Python to someone who has never programmed before?"

3. Why choose Python? (30 minutes)
- Python for beginners and professionals
- Advantages: simple syntax, large libraries, community, cross-platform
- Examples of Python in data science, machine learning, automation
- Interview-style question: "Why is Python a popular choice for beginners and for data analysis?"

4. Basic Python syntax (45 minutes)
- Python program structure: statements, indentation, comments
- Print statements and basic text output
- Python keywords and naming rules
- Example: write a simple script with `print()` and variables
- Interview-style question: "What are the basic syntax rules you must follow when writing Python code?"

Break (15 minutes)

5. Variables and assignment (45 minutes)
- What is a variable? How Python stores values
- Naming rules and conventions
- Assignment examples: `x = 10`, `name = "Alice"`
- Dynamic typing and variable reuse
- Interview-style question: "How do you declare variables in Python, and how are they different from variables in other languages?"

6. Data types in Python (60 minutes)
- Built-in types: `int`, `float`, `str`, `bool`, `list`, `tuple`, `dict`, `set`
- When to use each type
- Type checking with `type()` and conversion functions like `int()`, `float()`, `str()`
- Hands-on examples: math with numbers, text manipulation, lists and dictionaries
- Interview-style question: "What built-in data types does Python provide, and when would you use each?"

7. Operators in Python (45 minutes)
- Arithmetic operators: `+`, `-`, `*`, `/`, `//`, `%`, `**`
- Comparison operators: `==`, `!=`, `>`, `<`, `>=`, `<=`
- Logical operators: `and`, `or`, `not`
- Assignment operators: `=`, `+=`, `-=`
- Membership examples with `in` and `not in`
- Interview-style question: "What categories of operators exist in Python, and give examples of each?"

Break (15 minutes)

8. Practical mini project: sales analysis script (45 minutes)
- Create sample sales variables and a list of values
- Compute total and average sales
- Use operators to compare months and find the best month
- Print a simple summary report
- Connect the task to ADSA: data capture, calculation, interpretation

9. Review, Q&A, and homework (30 minutes)
- Review key concepts from Day 1
- Answer student questions
- Assign homework: practice problems on variables, data types, and operators
- Prepare for Day 2: control flow (`if`, `for`, `while`) and functions

Optional homework ideas:
- Write a program to calculate the average of three numbers
- Create a list of daily sales and print the maximum and minimum values
- Convert temperature from Celsius to Fahrenheit using variables and operators
