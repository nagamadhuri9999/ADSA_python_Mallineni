# Day 1: Python Basics, Variables, & Data Types Quiz

Test your knowledge of Day 1 concepts! Choose the best answer for each question.

---

### 1. Which of the following is a valid Python variable name?
A) `1st_number`
B) `first-number`
C) `first_number`
D) `first number`

### 2. What will the following code output?
```python
x = 10
y = "20"
print(x + y)
```
A) `30`
B) `1020`
C) A `TypeError`
D) `"30"`

### 3. Which data type is used to represent True or False values?
A) `int`
B) `str`
C) `bool`
D) `float`

### 4. What is the output of `15 // 4` in Python?
A) `3.75`
B) `3`
C) `4`
D) `15`

### 5. Which of the following collections is ordered and changeable (mutable)?
A) List
B) Tuple
C) Set
D) None of the above

---
<details>
<summary><b>Click here to view the answers</b></summary>

1. **C) `first_number`** (Variable names cannot start with a number, contain hyphens, or contain spaces. `snake_case` is preferred in Python).
2. **C) A `TypeError`** (You cannot add an integer (`10`) directly to a string (`"20"`). You must convert one of them first, e.g., `str(x) + y` or `x + int(y)`).
3. **C) `bool`** (Booleans represent logical states: `True` or `False`).
4. **B) `3`** (`//` is the floor division operator. It divides and rounds down to the nearest whole number).
5. **A) List** (Lists are ordered, mutable collections. Tuples are ordered but immutable. Sets are unordered and unindexed).
