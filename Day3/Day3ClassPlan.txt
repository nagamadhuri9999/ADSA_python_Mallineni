Day 3: Advanced Control, Structures, and Functions (6-hour online class)

Session goal:
Refine loop control, handle errors gracefully, deeply understand string/list slicing, and introduce the power of reusable functions and recursion.

Class structure:

1. Welcome and Review (30 minutes)
- Recap Day 2: `if`/`else`, `for`, and `while` loops
- Review the star pattern homework
- Today's objective: structuring our code with functions and manipulating sequences.

2. Loop Control Statements (`break`, `continue`, `pass`) (45 minutes)
- `break`: Exiting a loop early when a condition is met
- `continue`: Skipping the current iteration and moving to the next
- `pass`: Using a placeholder to avoid syntax errors in empty blocks
- Example: Searching for a specific value and breaking early
- Interview-style question: "What is the specific difference between `continue` and `pass` inside a loop?"

3. Error Handling and Exceptions (30 minutes)
- Introduction to the `try`/`except` block
- Common errors: `IndexError`, `ValueError`, `TypeError`
- How to prevent crashes and provide user-friendly error messages

Break (15 minutes)

4. String and List Operations & Slicing (60 minutes)
- Deep dive into sequence structures (both strings and lists)
- Positive and Negative Indexing (e.g., `list[-1]`)
- Slicing syntax: `[start:stop:step]`
- String/List concatenation (`+`) and repetition (`*`)
- Interview-style question: "How would you reverse a string or list using only slicing syntax?"

5. Introduction to Functions (60 minutes)
- Why use functions? Modularity and reusability
- The `def` keyword and syntax
- Returning values vs. printing values
- Positional and default arguments
- Example: Writing a robust mathematical function
- Interview-style question: "Explain the difference between arguments and parameters in a function."

Break (15 minutes)

6. Advanced Function Parameters and Variable Scope (45 minutes)
- Using `*args` to accept any number of positional arguments
- Using `**kwargs` to accept any number of keyword arguments
- Local vs. Global scope: When can a function see a variable?
- The `global` keyword (and why to avoid it)

7. Introduction to Recursion (30 minutes)
- What is recursion? (A function calling itself)
- The absolute necessity of the Base Case
- The Recursive Step
- Example: A simple countdown function

8. Review, Q&A, and Homework (30 minutes)
- Review key concepts from Day 3
- Answer student questions
- Assign homework: Test-driven practice problems on slicing and functions
- Prepare for Day 4: String methods, Anagrams, and advanced recursion dry runs.
