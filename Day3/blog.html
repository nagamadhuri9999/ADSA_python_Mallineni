1. Sequence Indexing

Both strings and lists are ordered sequences. Every item has a specific position, called an index. Python uses 0-based indexing.

3. Functions (`def`)

Functions are reusable blocks of code. You define them once, and call them anywhere.

flowchart LR A[Inputs / Arguments] --> B(Function Logic) B --> C[Output / Return Value] 

def greet(name, greeting="Hello"): 
    return f"{greeting}, {name}!"

print(greet("Alice"))               # Output: "Hello, Alice!"
print(greet("Bob", "Good morning")) # Output: "Good morning, Bob!"
