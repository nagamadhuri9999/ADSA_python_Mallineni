## What you will learn

- How to use `break`, `continue`, and `pass` in loops.
- What an `IndexError` is and how to avoid it.
- How to perform indexing and slicing on Strings and Lists.
- How to concatenate and repeat strings and lists.
- How to define and call functions in Python.
- Understanding arguments, parameters, `*args`, and `**kwargs`.
- How variable scope (local vs global) works.
- What recursion is and why a base case is necessary.

## How to use these files
