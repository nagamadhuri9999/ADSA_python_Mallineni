# Day 3: Strings, Lists, and Functions

Welcome to Day 3 of the Python Mastery Course! Today we focus on breaking data apart (slicing) and bundling logic together (functions).

## 🎯 Today's Topics

1. **Sequence Indexing & Slicing**
   - Extract exactly what you need using positive and negative indices.
2. **Functions (`def`)**
   - Write reusable, modular code blocks.
3. **Parameters and Arguments**
   - Default arguments, `*args` (Tuples), and `**kwargs` (Dictionaries).
4. **Variable Scope**
   - Understand the difference between `local` and `global` variables.
5. **Recursion**
   - The mind-bending concept of functions calling themselves (and how to stop them with Base Cases).
