2. Loop Control Statements: `break`, `continue`, `pass`
-------------------------------------------------------
- `break`: Exits the loop entirely. The loop is terminated immediately.
- `continue`: Skips the current iteration and moves to the next iteration of the loop.
- `pass`: A null statement. It does nothing and is used as a placeholder when a statement is syntactically required but you have no code to execute.

Example:
```python
for i in range(1, 10):
    if i == 5:
        break     # Loop stops when i is 5
    if i % 2 == 0:
        continue  # Skips even numbers
    print(i)
```

Example of `pass`:
```python
if True:
    pass # To be implemented later
```

3. IndexError
-------------
- An `IndexError` occurs when you try to access an index that is out of range for a sequence like a string or list.
Example:
```python
name = "Python"
print(name[10]) # Raises IndexError because "Python" only has indices 0 to 5.
```

4. String Operations
--------------------
- Strings are ordered sequences of characters.
- Indexing: Accessing a single character. Python uses 0-based indexing.
  - Positive indexing: starts from 0 (left to right).
  - Negative indexing: starts from -1 (right to left).
- Slicing: Extracting a substring using `[start:stop:step]`.
  - `start` is inclusive, `stop` is exclusive.
  - Positive step (+): You move forward. For output, `start` MUST be less than `stop` (`start < stop`), otherwise you get an empty string.
  - Negative step (-): You move backward. For output, `start` MUST be greater than `stop` (`start > stop`), otherwise you get an empty string.
- Concatenation: Joining strings using the `+` operator.
- Repetition: Repeating strings using the `*` operator.

Examples:
```python
text = "Python"
# Indexing
print(text[0])   # 'P'
print(text[-1])  # 'n'

# Slicing
print(text[0:3]) # 'Pyt' (Positive step, 0 < 3 -> Output)
print(text[3:0]) # '' (Positive step, 3 > 0 -> Empty Output)
print(text[3:0:-1]) # 'oht' (Negative step, 3 > 0 -> Output)
print(text[0:3:-1]) # '' (Negative step, 0 < 3 -> Empty Output)
print(text[::-1])# 'nohtyP' (Reverses string)

# Concatenation and Repetition
print("Hello" + " " + "World")
print("Hi " * 3)
```

5. List Operations
------------------
- Lists are ordered, mutable collections of items.
- Like strings, lists support positive/negative indexing and slicing.
- Concatenation is done using `+` to join two lists.

Examples:
```python
my_list = [10, 20, 30, 40, 50]
# Indexing
print(my_list[1])   # 20
print(my_list[-2])  # 40

# Slicing
print(my_list[1:4]) # [20, 30, 40]

# Concatenation
list2 = [60, 70]
print(my_list + list2) # [10, 20, 30, 40, 50, 60, 70]
```

6. Functions
------------
- A function is a block of reusable code that runs when called.
- Defined using the `def` keyword.
- Parameters: Variables listed inside the parentheses in the function definition.
- Arguments: Values sent to the function when it is called.

Syntax:
```python
def function_name(parameters):
    # code
    return value
```

- Positional arguments: Passed in the correct positional order.
- Default arguments: Provide a default value if no argument is passed.
- `*args`: Allows passing a variable number of positional arguments (as a tuple).
- `**kwargs`: Allows passing a variable number of keyword arguments (as a dictionary).

Examples:
```python
# Default arguments
def greet(name, msg="Hello"):
    print(f"{msg}, {name}")

greet("Alice")          # Hello, Alice
greet("Bob", "Hi")      # Hi, Bob

# *args and **kwargs
def show_info(*args, **kwargs):
    print(args)         # Tuple of positional args
    print(kwargs)       # Dictionary of keyword args

show_info(1, 2, 3, name="Alice", age=25)
```

7. Variable Scope
-----------------
- Scope determines the accessibility of variables.
- Local scope: Variables created inside a function are local to that function.
- Global scope: Variables created outside any function are global and can be accessed anywhere.
- Use the `global` keyword to modify a global variable inside a function.

Example:
```python
x = 10 # Global variable

def my_func():
    y = 5 # Local variable
    print(x, y)

my_func()
# print(y) # Would raise NameError because y is local to my_func
```

8. Recursion
------------
- A function that calls itself is a recursive function.
- Must have a base case to stop the recursion, otherwise it results in a stack overflow (RecursionError).

Example: Factorial
```python
def factorial(n):
    if n == 1 or n == 0:  # Base case
        return 1
    else:                 # Recursive step
        return n * factorial(n - 1)

print(factorial(5)) # 120
```
