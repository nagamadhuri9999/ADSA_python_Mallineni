Day 6 Notes: Time and Space Complexity (Big O Notation)

1. What is Big O Notation?
--------------------------
Big O Notation is the mathematical language used in computer science to describe the performance or complexity of an algorithm. 
Instead of measuring speed in seconds (which changes depending on whether you have a fast or slow computer), Big O measures how the number of operations GROWS as the input size (N) grows.

We always focus on the WORST-CASE scenario.

2. Common Time Complexities (From fastest to slowest)
-----------------------------------------------------

A. O(1) - Constant Time
The algorithm takes the exact same amount of time regardless of how much data there is.
Example: Accessing an element in an array by its index `arr[5]`.

B. O(log N) - Logarithmic Time
The algorithm halves the dataset in each step. Incredibly fast for large datasets.
Example: Finding a word in a dictionary, or Binary Search.

C. O(N) - Linear Time
The time it takes grows directly in proportion to the input size. If the array has 10 elements, it takes 10 steps. If 100 elements, 100 steps.
Example: A standard `for` loop iterating through a list.

D. O(N^2) - Quadratic Time
The time grows exponentially. If the array has 10 elements, it takes 100 steps. If it has 100 elements, it takes 10,000 steps!
Example: Nested loops (a `for` loop inside a `for` loop) over the same data.

3. The Rules of Big O
---------------------
Rule 1: Drop the constants. 
If you have two non-nested loops, it takes O(N) + O(N) = O(2N). We drop the 2, so it is just O(N).

Rule 2: Drop non-dominant terms.
If an algorithm has an O(N^2) step and an O(N) step, O(N^2) + O(N) becomes simply O(N^2). We only care about the bottleneck.

4. Space Complexity
-------------------
Space complexity measures how much EXTRA memory an algorithm uses relative to the input size.
- O(1) Space: You use a few variables (like `count` or `total_sum`) but you don't create new arrays that grow.
- O(N) Space: You create a brand new list or dictionary that copies the input data or grows alongside it.
