Time & Space Complexity

Today's Topics

  • What is Big O Notation?
  • O(1) Constant Time
  • O(N) Linear Time
  • O(N^2) Quadratic Time
  • O(log N) Logarithmic Time
  • Space Complexity (Memory vs Time)
  • Dropping Constants and Non-Dominant Terms

Key Concepts

1. The Bottleneck Rules

Big O only cares about the worst-case scenario and massive datasets.

  • O(2N) becomes O(N): Constants are ignored.
  • O(N^2 + N) becomes O(N^2): Smaller, non-dominant terms are dropped.

2. Visualizing Complexities

# O(1) Constant
def get_first(arr):
    return arr[0]

# O(N) Linear
def print_all(arr):
    for a in arr: print(a)

# O(N^2) Quadratic
def nested_loop(arr):
    for i in arr:
        for j in arr:
            print(i, j)
