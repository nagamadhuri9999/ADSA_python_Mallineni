# Day 7: Space Complexity & Sorting Algorithms

## 1. Space Complexity
- Understanding Auxiliary Space vs Input Space.
- O(1) Constant Space (In-place algorithms).
- O(n) Linear Space (Creating new arrays/data structures).

## 2. Array/List Operations & Traversal
- Finding Max, Min, Second Max, Second Min.
- Index-based extraction (Even indices, square/cube indices).
- Validation: Checking if an array is sorted.

## 3. Elementary Sorting Algorithms
- **Bubble Sort:** Repeatedly swapping adjacent elements (O(n^2)).
- **Selection Sort:** Finding the minimum element and placing it at the front (O(n^2)).
- **Insertion Sort:** Building a sorted array one element at a time (O(n^2)).
- Time and Space Complexity Analysis of sorting algorithms.
