1. Space Complexity Basics

O(1) Constant Space: The algorithm modifies the list in-place without creating any new data structures. All of our basic sorting algorithms today use O(1) space!

2. Visualizing Bubble Sort

Bubble Sort compares adjacent elements and swaps them if they are out of order. The largest elements "bubble" to the end.

def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        if not swapped: break # Early exit if sorted

3. Selection Sort vs Insertion Sort

Algorithm Strategy Time Complexity
Selection SortFind minimum, swap to front.O(n^2) Always
Insertion SortTake next item, insert into sorted part.O(n^2) Worst, O(n) Best
