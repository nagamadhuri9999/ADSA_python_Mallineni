Key Concepts

1. Linear Search O(N)

Checks every element one by one. Good for small or unsorted data.

def linear_search(arr, target):
    for i in range(len(arr)):
        if arr[i] == target: return i
    return -1

2. Binary Search O(log N)

MUST BE SORTED! Cuts the search space in half every iteration.

def binary_search(arr, target):
    low, high = 0, len(arr) - 1
    while low <= high:
        mid = (low + high) // 2
        if arr[mid] == target: return mid
        elif arr[mid] < target: low = mid + 1
        else: high = mid - 1
    return -1
