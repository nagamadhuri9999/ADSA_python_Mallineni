# Sorting Visualizer Academy

A God-Level educational platform for visualizing sorting algorithms, designed specifically for interview preparation and deep conceptual understanding.

## Live Demo
🚀 **[https://sorting-visualizer-academy.vercel.app](https://sorting-visualizer-academy.vercel.app)**

## Features
- **Algorithms Supported:**
  - Merge Sort (Out-of-place Array Extension)
  - Quick Sort (Lomuto Partitioning)
- **Visualizations:**
  - Synchronized Call Stack tracking variables and parameters
  - Live Recursion Tree branching and active states
  - Array Bar Visualization with specialized coloring (Active, Sorted, Pivot, Pointers)
- **UI/UX:**
  - Interactive playback controls (Step forward/backward, Play/Pause, Speed slider)
  - Dynamic IDE-style Code Panel highlighting the exact active execution line
  - Dark/Light Mode with a glassmorphic aesthetic

## Tech Stack
- React 19 + TypeScript + Vite
- Zustand (State Management & Trace Generation)
- Reactflow (Recursion Tree Visualization)
- Framer Motion (Smooth array bar animations)
- Tailwind CSS v4
- React Resizable Panels (IDE layout)
