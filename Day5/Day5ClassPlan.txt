Day 5: Advanced Lists, Inputs, and Problem Solving (6-hour online class)

Session goal:
Master list manipulation using methods, learn to read competitive programming style inputs efficiently, and tackle classical iteration problems (frequencies, sums, and combination pairs).

Class structure:

1. Welcome and Review (30 minutes)
- Recap Day 4: String methods and Recursion trees
- Address questions from the string algorithms and anagram homework
- Today's objective: deep dive into list operations and problem-solving patterns.

2. Reviewing String vs List Slicing (45 minutes)
- Reinforce that slicing mechanics `[start:stop:step]` apply to both.
- Revisit positive and negative indexing.
- Reversing a list using `[::-1]`.
- Interview-style question: "Why does reversing a list via slicing create a new object, whereas `.reverse()` modifies it in place?"

3. List Methods Deep Dive (60 minutes)
- Adding elements: `.append()`, `.insert()`, `.extend()`
- Removing elements: `.remove()`, `.pop()`, `.clear()`
- Utility methods: `.count()`, `.index()`, `.sort()`, `.reverse()`
- Mutating lists in place vs returning new lists.
- Interview-style question: "What is the specific difference between `.append()` and `.extend()`?"

Break (15 minutes)

4. Reading Different Types of Inputs (45 minutes)
- Understanding `input()` always returns strings.
- Reading standard single inputs: `n = int(input())`
- Reading space-separated multiple inputs: `x, y = map(int, input().split())`
- Converting input streams directly into lists: `arr = list(map(int, input().split()))`

5. Iteration Problem Solving: Sums and Products (45 minutes)
- Summing all numbers in a list using an accumulator variable.
- Finding the product of all numbers (handling 0 gracefully).
- Calculating the sum of squares.

Break (15 minutes)

6. Advanced Problem Solving: Frequencies and Pairs (60 minutes)
- Frequency counting: Iterating through a list and updating a dictionary tally.
- Nested loops for pairs: Finding all unique combinations of two elements in an array.
- Avoiding duplicate pairs by managing inner loop indices (`for j in range(i+1, n)`).

7. Review, Q&A, and Homework (30 minutes)
- Review key concepts from the entire 5-day module.
- Answer final questions on lists and inputs.
- Assign homework: Test-driven practice problems and final quiz.
- Closing thoughts on ADSA Python Fundamentals.
