1. List Memory References

Lists are mutable. When you assign a list to a new variable, or pass it into a function, you are passing a reference to the memory address. Modifying the new variable modifies the original list!

2. Combination Pairs Dry Run

To generate unique pairs from a list, use nested loops where the inner loop starts at i + 1.


letters = ["A", "B", "C", "D"]
for i in range(len(letters)):
    for j in range(i + 1, len(letters)):
        print(f"({letters[i]}, {letters[j]})")
