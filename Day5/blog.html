Key Concepts

1. List Methods Deep Dive

Python lists come with powerful built-in methods. Understanding which methods modify the list in-place versus which return a new object is crucial.

nums = [10, 20, 30]
nums.append(40)      # [10, 20, 30, 40]
nums.insert(1, 15)   # [10, 15, 20, 30, 40]
nums.extend([50, 60]) # Adds multiple elements
nums.pop()           # Removes and returns last element

2. Efficient Input Reading

For data analysis or competitive programming, reading inputs cleanly is a standard pattern.

# Read a space-separated list of numbers
# Input: 1 2 3 4 5
nums = list(map(int, input().split()))

3. Frequency Counting

Using dictionaries to count occurrences of elements in a list is a highly efficient O(N) operation.

freq = {}
for num in nums:
    freq[num] = freq.get(num, 0) + 1

1. What is the difference between `.append()` and `.extend()`?

  • A) They are identical
  • B) `.append()` adds a single element; `.extend()` adds elements from an iterable
  • C) `.extend()` adds a single element; `.append()` adds elements from an iterable
  • D) `.append()` is for strings, `.extend()` is for lists
View Answer

Answer: B

2. What does `list[::-1]` do?

  • A) Sorts the list in descending order
  • B) Returns a reversed copy of the list
  • C) Modifies the list in-place to reverse it
  • D) Returns the last element
View Answer

Answer: B (Slicing with a negative step reverses the sequence)

3. How do you remove the first occurrence of the value `20` from a list?

  • A) list.pop(20)
  • B) list.delete(20)
  • C) list.remove(20)
  • D) del list[20]
View Answer

Answer: C (`.remove()` removes by value, `.pop()` removes by index)

4. When finding unique pairs in a list using nested loops, the inner loop index `j` usually starts at:

  • A) 0
  • B) i
  • C) i + 1
  • D) len(list)
View Answer

Answer: C (Starting at `i+1` avoids duplicate pairs like (A,B) and (B,A))

5. What is the output of `map(int, ["1", "2", "3"])` when converted to a list?

  • A) ["1", "2", "3"]
  • B) [1, 2, 3]
  • C) Error
  • D) "123"
View Answer

Answer: B
