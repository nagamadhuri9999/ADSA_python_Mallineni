# Day 5: List Methods and Algorithmic Problem Solving

Welcome to Day 5! Today we master Python's most versatile data structure: the List. We will explore its built-in methods and use it to solve fundamental algorithmic problems.

## 🎯 Today's Topics

1. **List Methods**
   - Mutating lists in-place (`.append()`, `.remove()`, `.sort()`, `.reverse()`).
2. **Memory References**
   - Understanding how passing lists to functions or assigning them affects the original object.
3. **Input Processing**
   - Reading strings, splitting them, and mapping them to integers.
4. **Algorithmic Patterns**
   - Frequency Counting (O(N)).
   - Finding Combination Pairs (O(N^2)).
