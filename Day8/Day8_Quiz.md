# Day 8 Quiz: Advanced Sorting Algorithms

1. What algorithmic paradigm do both Merge Sort and Quick Sort use?
   a) Dynamic Programming
   b) Greedy Algorithms
   c) Divide and Conquer
   d) Backtracking
   *Answer: c*

2. What is the worst-case time complexity of Quick Sort?
   a) O(N)
   b) O(N log N)
   c) O(N^2)
   d) O(log N)
   *Answer: c*

3. Which of the following is true about Merge Sort?
   a) It sorts in-place.
   b) It has an O(N) space complexity.
   c) It is an unstable sorting algorithm.
   d) Its worst-case time complexity is O(N^2).
   *Answer: b*

4. If an algorithm preserves the relative order of elements with equal keys, it is called:
   a) Fast
   b) In-place
   c) Recursive
   d) Stable
   *Answer: d*

5. Which of the following algorithms is generally NOT stable?
   a) Merge Sort
   b) Bubble Sort
   c) Insertion Sort
   d) Quick Sort
   *Answer: d*
