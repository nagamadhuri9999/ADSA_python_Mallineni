# Day 8 Class Plan: Advanced Sorting Algorithms (6 Hours)

## Hour 1: The Divide & Conquer Paradigm and Merge Sort
- Concept of Divide and Conquer.
- Introduction to Merge Sort.
- Visualizing the merging process.
- Step-by-step code walkthrough.

## Hour 2: Quick Sort and Partitioning
- Introduction to Quick Sort.
- The Partitioning logic (choosing a pivot, rearranging elements).
- Step-by-step code walkthrough.
- Why Quick Sort is often preferred over Merge Sort for arrays.

## Hour 3: Complexity Analysis and Sorting Comparison
- Time Complexity (Best, Worst, Average) for Bubble, Selection, Insertion, Merge, and Quick Sort.
- Space Complexity (In-place vs. Out-of-place).
- Stability in sorting algorithms (Stable vs. Unstable).
- Real-world applications: When to use which algorithm?

## Hour 4: Hands-on Practice & LeetCode (Part 1)
- Implementing custom sorting logic using Python's built-in `sort()` and `sorted()`.
- Problem Solving: Sort Colors (LeetCode 75).
- Problem Solving: Kth Largest Element in an Array (LeetCode 215).

## Hour 5: Hands-on Practice & LeetCode (Part 2)
- Problem Solving: Merge Intervals (LeetCode 56).
- Activity: Write a script to compare the execution times of Bubble Sort vs Merge Sort.

## Hour 6: Mock Interviews and Q&A
- Rapid-fire conceptual questions (e.g., "What is the worst-case scenario for Quick Sort?").
- Live coding interview simulation.
- Review of tasks and wrap-up.
