Day 4: String Methods, Algorithms, and Advanced Recursion (6-hour online class)

Session goal:
Elevate string manipulation skills using built-in methods, apply algorithmic thinking to solve the classic Anagram problem, and demystify the call stack with recursion dry runs and mind mapping.

Class structure:

1. Welcome and Review (30 minutes)
- Recap Day 3: slicing, functions, and the basics of recursion
- Discuss the homework on slicing and functions
- Today's focus: built-in power tools for strings and analyzing recursive execution.

2. String Methods Deep Dive (60 minutes)
- Case manipulation: `.lower()`, `.upper()`, `.capitalize()`, `.title()`
- Cleaning text: `.strip()`, `.lstrip()`, `.rstrip()`
- Search and count: `.find()`, `.count()`
- Transformation: `.replace()`
- Splitting and Joining: `.split()`, `.join()` (Crucial for parsing data)
- Interview-style question: "How would you convert a comma-separated string into a list of integers?"

3. Problem Solving: Identifying Anagrams (45 minutes)
- What is an anagram? (e.g., 'listen' vs 'silent')
- Approach 1: Sorting. Why is `sorted(s1) == sorted(s2)` an easy but sometimes slower solution?
- Approach 2: Frequency Counting using Dictionaries. Why is this faster for large strings?
- Interview-style question: "Can you explain the time complexity difference between solving anagrams via sorting vs frequency counting?"

Break (15 minutes)

4. Recursion in More Detail (60 minutes)
- Re-establishing the Base Case and Recursive Step
- Visualizing the Call Stack
- What happens when a function calls itself? Where does the memory go?
- The concept of Stack Overflow

5. Recursion Dry Run: Factorial (45 minutes)
- Writing a recursive factorial function
- Manually tracing the execution frame-by-frame
- Seeing how return values bubble back up the stack

Break (15 minutes)

6. Mind Mapping a Recursion Tree: Fibonacci (60 minutes)
- Writing a recursive Fibonacci function
- Why is naive recursive Fibonacci slow?
- Drawing the recursion tree for `fib(4)`
- Visualizing redundant calculations and introducing the concept of optimization (Memoization)

7. Review, Q&A, and Homework (30 minutes)
- Review key concepts from Day 4
- Answer student questions
- Assign homework: Quiz and interview questions on string methods and recursion trees
- Prepare for Day 5: List methods, input reading, and list problem solving.
