1. Solving Anagrams

An anagram is a word formed by rearranging the letters of another word. The two main ways to solve anagram algorithms are Sorting and Frequency Counting.


# Approach 1: Sorting (O(n log n))
def is_anagram_sort(s1, s2):
    return sorted(s1) == sorted(s2)

# Approach 2: Frequency Counting (O(n))
def is_anagram_count(s1, s2):
    if len(s1) != len(s2): return False
    count = {}
    for char in s1:
        count[char] = count.get(char, 0) + 1
    for char in s2:
        if count.get(char, 0) == 0: return False
        count[char] -= 1
    return True
