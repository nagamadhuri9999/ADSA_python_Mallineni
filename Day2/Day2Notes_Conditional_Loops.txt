Day 2 Notes: Python Control Flow and Loops

1. Review from Day 1
-------------------
- Python syntax uses indentation to define blocks.
- Variables store values by name, and Python infers the data type automatically.
- Data types include `int`, `float`, `str`, `bool`, and collections like `list` and `dict`.
- Operators allow math, comparisons, and logic.

2. Decision Making: `if`, `elif`, `else`
----------------------------------------
- `if` checks a condition and runs code only when the condition is True.
- `elif` provides additional conditions when the first `if` is False.
- `else` runs when all previous conditions are False.

Syntax:
```python
if condition:
    # code when condition is True
elif other_condition:
    # code when other_condition is True
else:
    # code when none of the conditions are True
```

Example:
```python
score = 85
if score > 90:
    print("A")
elif score > 80:
    print("B")
elif score > 70:
    print("C")
elif score > 60:
    print("D")
elif score > 50:
    print("E")
else:
    print("Try better next time")
```

Real-world use:
- Decide eligibility for a program, grade a student, or choose a discount rate.

3. Nested `if else`
--------------------
- Nested `if` means placing an `if` or `else` block inside another `if` block.
- Useful for checking multiple levels of conditions.

Example:
```python
number = 15
if number > 0:
    if number % 2 == 0:
        print("Positive even")
    else:
        print("Positive odd")
else:
    print("Zero or negative")
```

4. Loops: repeat logic multiple times
-------------------------------------
- Loops run the same block of code many times.
- Use loops when you want to repeat similar operations.
- Two main loop types in Python: `for` and `while`.

5. `for` loop
--------------
- Best when you know how many times to repeat.
- Often used with `range()`, strings, lists, and other sequences.

Syntax:
```python
for variable in sequence:
    # code to repeat
```

Example:
```python
for i in range(1, 11):
    print(i)
```

`range(start, stop, step)`:
- `start` is inclusive, `stop` is exclusive.
- `step` is the amount to move each time.
- Example: `range(100, 0, -5)` counts down from 100 to 5.

Example with strings:
```python
name = "Venkatesh"
for letter in name:
    print(letter)
```

6. `while` loop
----------------
- Use when the number of repetitions depends on a condition.
- The loop continues until the condition becomes False.

Syntax:
```python
while condition:
    # code to repeat
```

Example:
```python
num = 1
while num <= 10:
    print(num)
    num += 1
```

Countdown example:
```python
num = 10
while num >= 1:
    print(num)
    num -= 1
```

Real-world use:
- Keep asking for input until the user enters a valid value.
- Continue processing until an external limit is reached.

7. Nested loops
----------------
- A loop inside another loop is called a nested loop.
- The inner loop runs completely for each iteration of the outer loop.

Example:
```python
for i in range(1, 4):
    for j in range(1, 4):
        print(i, j)
```

Use cases:
- Generating multiplication tables.
- Building patterns such as squares and triangles.
- Working with rows and columns in a grid.

8. Star patterns
----------------
- Star patterns are a great way to practice loops and nested loops.
- Use string repetition with `"* " * n` to print repeated symbols.

Example:
```python
for i in range(1, 5):
    print("* " * i)
```

Example of a full square pattern:
```python
for i in range(5):
    print("* " * 5)
```

9. FizzBuzz and logic practice
-------------------------------
- FizzBuzz is a classic coding exercise for control flow:
  - print `Fizz` if the number is divisible by 3
  - print `Buzz` if the number is divisible by 5
  - print `FizzBuzz` if divisible by both
  - otherwise print the number

Example:
```python
for num in range(1, 16):
    if num % 3 == 0 and num % 5 == 0:
        print("FizzBuzz")
    elif num % 3 == 0:
        print("Fizz")
    elif num % 5 == 0:
        print("Buzz")
    else:
        print(num)
```

10. Debugging and dry run
-------------------------
- Dry run means tracing the code by hand before running it.
- Check variable values and flow for each iteration.
- This helps find indentation issues and logic errors.

Example:
- For `while num >= 1`, check that `num` changes correctly each step.
- For `for i in range(1, 5)`, verify that the loop runs exactly 4 times.
