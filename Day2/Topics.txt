# Day 2: Control Flow (Conditionals & Loops)

1. Conditional Statements (if, elif, else)
   - Decision making based on boolean conditions.
   - Syntax and indentation rules for `if`, `elif`, and `else`.

2. Nested Conditionals
   - Chaining and embedding decisions to handle complex scenarios.
   - Example: Grading systems.

3. Introduction to Loops
   - Understanding iteration and why we use loops (DRY principle).
   - Differences between `for` and `while` loops.

4. The `for` Loop
   - Iterating over ranges, sequences (lists), and strings.
   - Using the `range()` function.

5. The `while` Loop
   - Executing blocks of code as long as a condition is true.
   - Managing state and avoiding infinite loops.

6. Nested Loops
   - Putting loops inside loops for multi-dimensional operations.
   - Ideal for grids, matrices, and pattern printing.

7. Loop Control Statements
   - `break`: Exiting a loop early.
   - `continue`: Skipping the current iteration.

8. Practical Exercise: Star Patterns
   - Using nested loops and string repetition to draw visual shapes in the console.
