Day 2: Control Flow & Loops

Mastering decisions and iteration in Python

1. Conditional Statements

Conditionals allow your code to execute different instructions based on boolean (True/False) conditions.

flowchart TD A[Start] --> B{Is condition True?} B -- Yes --> C[Execute 'if' block] B -- No --> D{Is next 'elif' True?} D -- Yes --> E[Execute 'elif' block] D -- No --> F[Execute 'else' block] C --> G[Continue] E --> G F --> G 

temperature = 25

if temperature > 30:
    print("It's hot outside.")
elif temperature > 20:
    print("The weather is nice.")
else:
    print("It's a bit cold.")

2. The For Loop & Dry Run

A for loop is used to iterate over a sequence (like a list, tuple, or using range()).


# Print even numbers from 0 to 4
for i in range(0, 5, 2):
    print(i)

Dry Run Analysis

Iteration Value of i Output State

3. Loop Control: Break & Continue

  • break: Instantly terminates the entire loop.
  • continue: Skips the rest of the current iteration and jumps to the next one.
