Day 2: Control Flow in Python (6-hour online class)

Session goal:
Master decision-making and repetition in Python. By the end of this session, students will be able to write programs that make choices using `if`/`else` and repeat actions using `for` and `while` loops, applying them to pattern generation.

Class structure:

1. Welcome and Review (30 minutes)
- Recap Day 1: variables, data types, and operators
- Answer any lingering questions from the homework
- Outline today's objectives: giving programs the ability to make choices and repeat themselves.

2. Conditional Statements (`if`, `elif`, `else`) (60 minutes)
- Introduction to boolean logic and decision branches
- The `if` statement syntax and indentation rules
- Handling multiple conditions with `elif` and `else`
- Example: Grading system based on student scores
- Interview-style question: "How do you handle multiple conflicting conditions in Python without writing nested if statements everywhere?"

3. Nested If-Else Statements (45 minutes)
- When and why to use nested conditionals
- Maintaining readability with proper indentation
- Example: Checking age and ID status for building access
- Common pitfalls: "dangling else" logic errors

Break (15 minutes)

4. Introduction to Loops (45 minutes)
- Why do we need loops? The DRY (Don't Repeat Yourself) principle
- The difference between definite (`for`) and indefinite (`while`) loops
- Interview-style question: "When would you choose a while loop over a for loop in a data analysis task?"

5. The `for` Loop (45 minutes)
- Iterating over ranges using `range()`
- Iterating over sequences (like strings)
- Using the loop variable to perform calculations
- Example: Generating multiplication tables

6. The `while` Loop (45 minutes)
- Syntax and condition checking
- Preventing infinite loops (updating the condition)
- Example: User input validation (keep asking until correct)

Break (15 minutes)

7. Nested Loops and Star Patterns (45 minutes)
- Putting loops inside loops: How the inner and outer loops interact
- Why nested loops are useful for grids, matrices, and coordinates
- Practical exercise: Printing a right-angled triangle and a square using star (`*`) characters
- Understanding `print("*", end="")` and newline management

8. Review, Q&A, and Homework (30 minutes)
- Review key concepts from Day 2
- Answer student questions
- Assign homework: practice problems on loops and conditionals
- Prepare for Day 3: Break/continue, error handling, lists, and functions

Optional homework ideas:
- Write a program that prints a full pyramid star pattern.
- Build a simple number guessing game using a `while` loop and `if`/`else` checks.
