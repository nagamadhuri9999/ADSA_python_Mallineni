# Day 10: Introduction to Stacks

Welcome to Day 10! Today is all about understanding the **Stack**, one of the most elegant and commonly used data structures in computer science.

## 🎯 Today's Topics

1. **The LIFO Principle**
   - Last In, First Out mechanics.
2. **Core Operations**
   - Push, Pop, Peek, and checking for Empty.
3. **Implementation in Python**
   - Leveraging Python's built-in `list` to achieve O(1) time complexity.
4. **Powerful Applications**
   - Simulation, Validating Parentheses, and Monotonic Sequences.
