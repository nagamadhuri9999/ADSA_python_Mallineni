Day 10: Linked Lists

Nodes, Pointers, and Dynamic Memory.

1. The Node Structure

Unlike Arrays where elements sit next to each other, Linked List elements (Nodes) are scattered in memory. A Node stores data AND a pointer to the next Node.


class Node:
    def __init__(self, data):
        self.data = data
        self.next = None

2. Visualizing a Linked List

The entire list is controlled by a single reference to the first node, called the Head.

3. Traversal

To print or search a Linked List, we must "walk" through it node by node using a loop. We cannot jump straight to index 5!


current = head
while current is not None:
    print(current.data)
    current = current.next

4. Insertions

Inserting at the Head is extremely fast O(1) because we don't need to shift any existing elements!


new_node = Node(5)
new_node.next = self.head  # Point new node to old head
self.head = new_node       # Update the head pointer
