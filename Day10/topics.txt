# Day 10: Introduction to Stacks

## 1. Stack Fundamentals
- LIFO (Last In, First Out) principle.
- Mental models: A stack of plates, a back button in a browser.

## 2. Basic Operations
- `push()`: Adding to the top.
- `pop()`: Removing from the top.
- `peek()` / `top()`: Viewing the top without removing.
- `isEmpty()`: Checking if the stack is empty.

## 3. Implementation in Python
- Using Python's built-in `list` as a stack.
- `append()` vs `push()`.
- `pop()` mechanics.
- Time Complexities (O(1) operations).

## 4. Powerful Applications
- String Reversals.
- Validating Parentheses.
- Evaluating Postfix / Reverse Polish Notation.
- Monotonic Stacks (Next Greater Element).
